## Customization

### Fonts
The site uses:
- **Cormorant** (serif) for headlines
- **Inter** (sans-serif) for body text

### Colors
Based on Tailwind's stone palette for that editorial feel:
- Background: `stone-50`
- Text: `stone-900`
- Accents: `stone-500`, `stone-600`

### Sections to Update

1. **Hero Section** (`#hero`)
   - Replace "Your Name" with your actual name
   - Update location and tagline
   - Add hero image

2. **About Section** (`#about`)
   - Update bio text
   - Customize expertise and tools lists

3. **Experience Section** (`#experience`)
   - Add your work history
   - Update company names and dates

4. **Projects Section** (`#projects`)
   - Replace placeholder projects
   - Add project images
   - Update descriptions and tech stacks

5. **Contact Section** (`#contact`)
   - Update email and phone
   - Add your social media links

### Interactive Elements

The site includes several micro-interactions:

- **Hero Grid**: Animated marquee, morphing shapes, and interactive grid cells
- **About Section**: Hover cards with emoji changes and pulsing dots
- **Projects**: Color-shifting cards, rotating arrows, ripple effects, and wave animations
- **3D Tilt**: Bento cards have subtle 3D parallax on hover
- **Click Interactions**: Various cards respond to clicks with animations

All interactions are built with vanilla JavaScript for performance.

## Deployment to GitHub Pages

1. Create a new repository on GitHub
2. Push your code:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/yourusername/your-repo.git
   git push -u origin main
   ```

3. Enable GitHub Pages:
   - Go to repository Settings
   - Navigate to Pages section
   - Select "main" branch as source
   - Save

Your site will be live at: `https://yourusername.github.io/your-repo/`
